# 🔥 Fire Fighting Bot Website

A modern web interface showcasing a **Fire Fighting Robot system** built using ESP32.

---

## ⚙️ Tech Stack
- HTML5
- CSS3
- JavaScript (basic)
- ESP32

---

## 🤖 Features
- Fire detection simulation 🔥
- Modern dark UI 🎨
- Project explanation & structure

---

## 🧠 Working
Flame sensor detects fire → ESP32 processes signal → robot moves → water pump extinguishes fire.

---

## 🌍 Applications
- Smart homes  
- Industrial safety  
- Hazardous areas  

---

## 🚀 Run Project
1. Open `index.html`
2. Run with Live Server
